CSS
CSS (Cascading Style Sheets) es un lenguaje que controla el diseño y la presentación de páginas web, complementando el contenido básico de HTML. Permite definir reglas para el estilo de elementos como fuentes, colores y tamaños, que se pueden aplicar en el mismo documento HTML o en archivos separados, como style.css, mediante la referencia en la etiqueta head.

JAVASCRIPT
Java es un lenguaje de programación orientado a objetos de alto nivel, conocido por su portabilidad y capacidad de ejecutarse en diferentes plataformas. Esto se logra gracias a la Máquina Virtual de Java (JVM), que permite que los programas Java se ejecuten en cualquier dispositivo compatible con JVM.
